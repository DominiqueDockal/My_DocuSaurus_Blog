## Quickstart

1. Clone the starter template repository from Developer Akademie.
2. Create a dedicated feature branch named `setup-blog`.
3. Install all dependencies with `pnpm install`.
4. Start the local development server with `pnpm start`.
5. Adjust the Docusaurus configuration for title, tagline, deployment, navigation, footer, and repository links.
6. Update the README documentation.
7. Prepare the project for deployment via GitHub Pages and GitHub Actions.

## Description

This project is based on the Developer Akademie Docusaurus starter template and was customized to create a personal developer blog and learning journal.

During setup, the template repository was cloned and a dedicated feature branch called `setup-blog` was created for all personalization and configuration work. After installing dependencies, the project was tested locally to confirm that the development environment was working correctly.

The Docusaurus configuration was then adjusted to personalize the blog. This included updating the site title and tagline, configuring deployment-related environment variables, setting the repository URL, and updating the `editUrl` values for the docs and blog sections.

In addition, the navigation bar and footer were customized to include the correct project and repository links. The README file was also updated to document the repository structure, local setup, and deployment behavior.
